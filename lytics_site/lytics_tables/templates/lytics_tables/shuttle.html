Shuttle app

This app aims to solves the classical traveling salesmans problem. UCR provides a program in which they shuttle students to their requested destinations. The shuttle driver has to create a route based on all the requests provided by the students which may lead to a nonoptimal route. The following is a small demo of the traveling salesman problem.

Features in the development pipeline:

  • User groups (admin, shuttle driver, students)
  • Database
  • Analytics
  • Heatmap
  • Map Integration
